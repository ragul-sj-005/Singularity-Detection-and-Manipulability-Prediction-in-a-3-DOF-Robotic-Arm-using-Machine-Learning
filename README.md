# Singularity-Detection-and-Manipulability-Prediction-in-a-3-DOF-Robotic-Arm-using-Machine-Learning
Singularities in robots cause loss of control. This project models a 3-DOF arm, generates 100k+ configurations, and uses SVM/SVR to predict singularities and manipulability in real time. A hybrid ML+physics system ensures safe, fast robotic motion.
